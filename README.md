# Arrow War

> 화살로 몰려오는 적을 막아라! 나만의 화살을 강화하며 성을 지키는 캐슬 디펜스 게임.

---

## 게임 개요

**Arrow War**는 플레이어가 다양한 종류의 화살을 사용해 적의 공격으로부터 자신의 성을 지키는 2D 캐슬 디펜스 게임입니다.
화면 왼쪽의 아군 성을 지키면서, 오른쪽 적 성에서 쏟아져 나오는 몬스터들을 화살로 처치하세요.
골드를 모아 특수 화살을 해금하고, 총 3번의 매치를 모두 승리하면 게임에서 이깁니다.

---

## 게임 방식

1. 게임이 시작되면 화면 오른쪽 적 성에서 몬스터들이 지속적으로 스폰됩니다.
2. 몬스터들은 화면 왼쪽의 아군 성을 향해 이동하며, 성에 도달하면 성에 피해를 입힙니다.
3. 플레이어는 마우스 클릭으로 화살을 발사해 몬스터를 처치합니다.
4. 몬스터를 처치하면 골드를 획득하며, 골드로 특수 화살을 해금할 수 있습니다.
5. 한 매치는 여러 라운드로 구성되며, 모든 라운드를 클리어하면 업그레이드 패널이 열립니다.
6. 총 3번의 매치를 모두 승리하면 최종 승리입니다.

---

## 주요 시스템

### 화살 슬롯
- 화살은 슬롯 단위로 관리되며, 키보드 숫자키(1~5)로 원하는 슬롯을 선택합니다.
- 각 화살마다 고유한 쿨타임이 있으며, 슬롯 UI에서 쿨타임 진행 상황을 확인할 수 있습니다.
- 선택된 슬롯은 **노란색**, 미선택 슬롯은 **초록색**으로 표시됩니다.
- 쿨타임 중인 슬롯은 방사형(Radial) 채우기 이펙트로 남은 시간을 시각적으로 보여줍니다.

### 골드 시스템
- 화살로 몬스터를 처치하면 골드를 획득합니다.
- 몬스터 처치 시 머리 위에 획득한 골드량이 노란색 텍스트로 떠오릅니다.
- 획득한 골드는 매치 종료 후 업그레이드 패널에서 특수 화살을 구매하는 데 사용됩니다.

### 업그레이드 패널
- 한 매치의 모든 라운드를 클리어하면 업그레이드 패널이 표시됩니다.
- 패널에서는 보유하지 않은 특수 화살을 골드로 구매할 수 있습니다.
- 구매한 화살은 다음 매치부터 슬롯에 자동으로 추가됩니다.

## 전투 메커니즘

### 기본 화살

| 이름 | 특징 |
|------|------|
| 기본 화살 (Basic Arrow) | 단일 대상에게 피해를 줍니다. 게임 시작 시 기본 지급됩니다. |

### 특수 화살

| 이름 | 효과 |
|------|------|
| 불 화살 (Fire Arrow) | 착탄 지점 주변의 적에게 광역 피해를 줍니다. |
| 얼음 화살 (Ice Arrow) | 적중한 적의 이동 속도를 일정 시간 동안 감소시킵니다. |
| 독 화살 (Poison Arrow) | 착탄 지점에 독 구름을 생성하여, 구름 안의 적에게 지속적으로 피해를 줍니다. |

### 적 AI
- 몬스터는 스폰 후 아군 성을 향해 직선으로 이동합니다.
- 성에 도달하면 이동을 멈추고 일정 간격으로 성을 공격합니다.
- 몬스터가 사망하면 짧은 사망 애니메이션 후 제거됩니다.

---

## 승리 및 패배 조건

### 패배
- 아군 성의 HP가 0이 되면 즉시 패배합니다.
- 결과 화면에서 다시 시작 버튼으로 처음부터 재도전할 수 있습니다.

### 매치 클리어
- 한 매치의 모든 라운드에서 적을 전부 처치하면 매치를 클리어합니다.
- 매치 클리어 후 업그레이드 패널이 열리며 특수 화살을 해금할 수 있습니다.

### 최종 승리
- 총 **3번의 매치**를 모두 클리어하면 게임에서 최종 승리합니다.
- 승리 화면이 표시되며 게임이 종료됩니다.

---

## 조작 방법

| 입력 | 동작 |
|------|------|
| 마우스 왼쪽 클릭 | 클릭한 위치를 향해 현재 선택된 화살 발사 |
| 숫자키 1 ~ 5 | 해당 번호의 화살 슬롯 선택 |
